## ComputerRoom-Reservation-System
简易机房预约系统，实现管理员，老师，学生代表的注册，预约，审核~

### Environment 
vs2019


### 使用说明
用vs新建项目，将所有文件放在项目目录下，编译即可。
项目来源于B站黑马视频课，代码大约1000行，算是一个比较综合的项目，适合C++入门。
